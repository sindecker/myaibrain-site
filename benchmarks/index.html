Methodology

  • Recall@5 (Ra@5): fraction of queries where the correct session appears in the top 5 retrieved results
  • NDCG@5: ranking quality — whether correct results appear at higher positions
  • SelRoute: assigns queries to specialized retrieval pipelines (FTS5 / embedding / hybrid) based on predicted query type. No LLM at query time. No GPU required. Runs on commodity hardware.
  • Full methodology: Sindecker (2026). SelRoute: Selective Query-Type Routing for Long-Term Conversational Memory Retrieval. arXiv:2604.02431
